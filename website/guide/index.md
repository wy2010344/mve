# MVE 框架指南

欢迎阅读 MVE 框架的官方指南！本指南将帮助你了解 MVE 框架的核心概念、使用方法和最佳实践。

## 指南内容

### 入门

- [快速开始](./getting-started.md) - 搭建 MVE 项目环境并创建你的第一个应用
- [架构概览](./architecture-overview.md) - 了解 MVE 框架的核心架构和设计理念
- [API 对比表](./api-comparison-table.md) - 快速查找 Vue/React API 在 MVE 中的对应实现

### 进阶学习

掌握了基础后，你可以继续学习以下内容：

- **核心概念** - 深入理解 MVE 的响应式系统、生命周期和渲染机制
- **组件开发** - 学习如何构建可复用的 MVE 组件
- **动画与过渡** - 实现流畅的界面动画效果
- **路由管理** - 配置和使用 MVE 的路由系统
- **Canvas 开发** - 利用 MVE 的 Canvas 渲染能力构建复杂图形应用

## 谁适合使用 MVE？

- **前端开发者** - 特别是熟悉 Vue 或 React 的开发者可以快速上手
- **追求性能的项目** - MVE 的响应式系统设计注重性能优化
- **需要灵活架构的团队** - 模块化设计允许按需引入功能
- **学习前端框架原理的学习者** - MVE 的设计相对简洁，易于理解框架工作原理
