# MVE 框架

MVE是一个现代化的前端框架，采用信号驱动的响应式设计，让开发者能够轻松构建高性能、可维护的Web应用。

## 为什么选择 MVE?

- **简洁直观**：API设计简单明了，学习曲线平缓
- **高性能**：精确的依赖追踪和批量更新机制
- **灵活组合**：模块间低耦合，可按需使用
- **强大功能**：支持组件、动画、路由、Canvas等多种功能
