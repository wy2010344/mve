# MVE Framework Guide

Welcome to the official guide for MVE Framework! This guide will help you understand the core
concepts, usage methods, and best practices of the MVE framework.

## Guide Contents

### Getting Started

- [Quick Start](./getting-started.md) - Set up the MVE project environment and create your first
  application
- [Architecture Overview](./architecture-overview.md) - Understand MVE framework's core architecture
  and design philosophy
- [API Comparison Table](./api-comparison-table.md) - Quickly find the MVE equivalents of Vue/React
  APIs

### Advanced Learning

After mastering the basics, you can continue to learn the following content:

- **Core Concepts** - Deeply understand MVE's reactive system, lifecycle, and rendering mechanism
- **Component Development** - Learn how to build reusable MVE components
- **Animation & Transitions** - Implement smooth interface animation effects
- **Route Management** - Configure and use MVE's routing system
- **Canvas Development** - Utilize MVE's Canvas rendering capabilities to build complex graphical
  applications

## Who is MVE for?

- **Frontend Developers** - Especially developers familiar with Vue or React can quickly get started
- **Performance-focused Projects** - MVE's reactive system design focuses on performance
  optimization
- **Teams Needing Flexible Architecture** - Modular design allows for on-demand feature inclusion
- **Learners of Frontend Framework Principles** - MVE's design is relatively concise, making it easy
  to understand how frameworks work
